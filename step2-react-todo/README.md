# Step 2 - React + Bootstrap Todo App

This is a degraded Step 2 version of the semester project.

## What it includes
- React frontend
- Bootstrap styling
- json-server backend
- GET, POST, PATCH, DELETE functionality
- Edit/save feature for updating todo names

## Folder structure
- `backend/` -> json-server API
- `frontend/` -> React app built with Vite

## Run backend
```bash
cd backend
npm install
npm start
```
Backend runs at `http://localhost:5000`.

## Run frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend usually runs at `http://localhost:5173`.
